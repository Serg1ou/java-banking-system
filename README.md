## Description
This project implements a simplified banking system in Java, including:

- IBAN generation and validation

- Transaction history with fixed capacity
- Custom exceptions and validation rules

## Features
- Account creation with age validation
- Secure money transfers between banks
- Transaction lifecycle management (OPEN, CLOSED, CANCELLED)
- Detection and handling of outdated transactions
- Custom exception hierarchy
